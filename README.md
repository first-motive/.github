# .github

Organization-level defaults for the [First Motive](https://github.com/first-motive) org.

The org profile page rendered at [github.com/first-motive](https://github.com/first-motive)
is sourced from [`profile/README.md`](profile/README.md). Edit that file to
change what visitors see on the org landing page.

```
.github/
└── profile/
    └── README.md    org landing page (rendered at github.com/first-motive)
```
